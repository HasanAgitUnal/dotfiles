# My dotfiles 

A Hyprland rice with neon colors. UI is clean and everything is handmade.

*WM*: Hyprland
*Bar*: Waybar
*Menu*: Rofi
*Terminal*: Foot
*Shell*: Bash
*File Manager*: Yazi & Thunar
*Notification Daemon*: Dunst
*OSD*: Custom Script
*LM*: Greetd with Regreet
*GTK Theme*: Custom theme based on Materia.
*Icon Theme*: Custom theme based on Papirus.

> You can find GTK and Icon theme [here](https://github.com/HasanAgitUnal/CybrCyanThemes)

This repository also contains my some `/etc` configrations.
